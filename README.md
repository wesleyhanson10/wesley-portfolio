# Wesley Hanson - Portfolio Website

A modern, colorful portfolio website showcasing my experience in biomedical engineering, product development, and innovation.

## 📄 Pages

- **Home** - Section with featured project and quick stats
- **About** - Background, expertise, skills, and global upbringing
- **Projects** - Detailed project cards for HungerQs, L'Oréal, and research
- **Experience** - Timeline of internships, research, and career development
- **Resume** - Complete resume with education, skills, and work history

## 🎨 Features

- Fully responsive layout (mobile, tablet, desktop)
- Project showcases with detailed information
- Timeline of professional experience
- Global upbringing

## 🚀 Deployment

This portfolio is hosted on GitHub Pages:

Visit: `https://wesleyhanson10.github.io/wesley-portfolio/`

## 📝 To Update Content

1. Edit the respective `.html` files
2. Update styles in `css/style.css`
3. Add images to `assets/` folder
4. Commit and push to GitHub
5. Changes will be reflected on the live site

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- GitHub Pages
